## Core Database Concepts Demonstrated

This portfolio highlights practical mastery over standard relational database manipulation:

* **Data Definition & Schema Design (DDL):** Building clean table matrices utilizing explicit tracking constraints (`PRIMARY KEY`, `FOREIGN KEY`, `NOT NULL`, `AUTO_INCREMENT`).
* **Relational Table Joins:** Merging data fields across multiple logical boundaries using structured relational links (`INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`).
* **Aggregate Data Analysis:** Extracting summary insights from raw system datasets using arithmetic filters and statistical grouping structures (`COUNT`, `SUM`, `AVG`, `GROUP BY`, `HAVING`).
* **Advanced Row Filtering:** Isolating conditional profiles cleanly using conditional text matching and logic evaluation arrays (`WHERE`, `LIKE`, `IN`, `BETWEEN`, `ORDER BY`).
